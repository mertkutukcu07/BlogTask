# BlogTask
1- bağlı kütüphaneleri yükleyelim
npm install

2- Backendi ayağa kaldıralım.

json-server --watch db.json

3- Projeyi ayağa kaldıralım.
npx expo start

4- Androidde çalıştırmak için
npx expo start --android

5- iOSta çalıştırmak için
npx expo start --ios
